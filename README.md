
# IcePick-to-Katana-PAYDAY-2-MOD-

A PAYDAY 2 mod that replaces the Katana model with the Ice Pick skin.

## 📝 Description

This mod replaces the visual model and assets of the Katana melee weapon with the Ice Pick model in PAYDAY 2. It includes both the asset overrides and the script files needed for the game.

---

## 🛠️ Requirements

* [SuperBLT](https://superblt.znix.xyz/) (Required for Lua scripts)
* `mod_overrides` folder enabled in your PAYDAY 2 directory

---

## 📥 Installation

1. Download the repository or grab the `.zip` files directly from the main page:
   * `Put This In mod_overrides.zip`
   * `Put this in mods folder.zip`
2. **Extract both `.zip` archives.**
3. Place the extracted folders into your PAYDAY 2 installation directory:
   * Extract contents of `Put This In mod_overrides.zip` into `PAYDAY 2/assets/mod_overrides/`
   * Extract contents of `Put this in mods folder.zip` into `PAYDAY 2/mods/`
4. Launch PAYDAY 2 and check your inventory!
